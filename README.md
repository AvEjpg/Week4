
# Прогнозирование показателей деятельности предприятия

Система прогнозирования ежемесячного объема продаж на основе временных рядов (2021–2023 гг.). В проекте реализованы методы линейной регрессии, полиномиальной регрессии и ARIMA для анализа трендов и построения прогнозов.

## 📋 Функционал

- Загрузка и предобработка данных из CSV.
- Проверка сезонности и декомпозиция временного ряда.
- Оценка точности моделей по метрикам: MAE, MSE, RMSE, MAPE.
- Визуализация фактических данных и прогнозных кривых.
- Прогноз на будущие 6 месяцев.

## 🛠 Технологии

Проект написан на Python 3.12 с использованием следующих библиотек:

- `pandas` — обработка данных.
- `numpy` — математические вычисления.
- `matplotlib` — визуализация графиков.
- `scikit-learn` — реализация линейной и полиномиальной регрессии.
- `statsmodels` — ARIMA модель.

## 🚀 Запуск проекта

### 1. Требования

Убедитесь, что у вас установлен Python. Установить необходимые библиотеки можно одной командой:

```bash
pip install pandas numpy matplotlib scikit-learn statsmodels
```

### 2. Подготовка данных

Файл с данными `PP13_ISP22_timeseries.csv` должен находиться в той же директории, что и скрипт. Формат файла: `date,monthly_sales`.

### 3. Запуск

Запустите Jupyter Notebook или выполните скрипт:

```bash
jupyter notebook main.ipynb
```
